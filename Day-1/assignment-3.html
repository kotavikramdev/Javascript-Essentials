Javascript Datatypes

In JavaScript there are two different kinds of data: primitives, and objects.

Primitive Datatypes: A primitive is simply a data type that is not an object, and has no methods.

There are Six Primitive datatypes:

  • Boolean
  • Number
  • String
  • Null
  • Undefined
  • Symbol

Object Datatypes: An object is a collection of properties. These properties are stored in key/value pairs. Properties can reference any type of data, including objects and/or primitive values.
